💡 TIPS DE MANTENIMIENTO
─────────────────────────────────────────────────────────────────────────────

1. Para cambiar colores:
   Editar en templates/base.html (líneas 10-19):
   ```css
   :root {
       --mundial-red: #E63946;        ← Cambiar aquí
       --mundial-blue: #1D3557;       ← Cambiar aquí
   }
   ```

2. Para agregar más templates:
   • Incluir {% extends "base.html" %}
   • Usar las clases CSS existentes
   • Mantener consistencia de colores
   • Agregar iconos con Font Awesome

3. Para optimizar velocidad:
   • Los estilos ya están incrustados
   • No agregar más CSS en archivos separados
   • Usar clases Bootstrap existentes
   • Minimizar JavaScript innecesario
